# Vector Add Implementation Guide

## Task 1: Implement tl_mul_relu_1d

### Reference tl_add_1d Structure

```python
@tilelang.jit
def tl_mul_relu_1d(A, B, BLOCK_N: int):
    N = T.const("N")
    A: T.Tensor((N,), T.float16)
    B: T.Tensor((N,), T.float16)
    C = T.empty((N,), T.float16)

    with T.Kernel(N // BLOCK_N, threads=256) as bx:
        base_idx = bx * BLOCK_N
        for i in T.Parallel(BLOCK_N):
            # Step 1: Calculate multiplication
            result = A[base_idx + i] * B[base_idx + i]
            # Step 2: Apply ReLU (max(0, result))
            C[base_idx + i] = T.if_then_else(result > 0, result, T.float16(0))

    return C
```

### Key Points

1. **Kernel Structure**: Exactly the same as addition
2. **Multiplication Operation**: `A[i] * B[i]` is element-wise multiplication
3. **ReLU Implementation**: `T.if_then_else(condition, true_value, false_value)`
   - condition: `result > 0`
   - true_value: `result`
   - false_value: `T.float16(0)` or just `0`

## Task 2: Implement tl_mul_relu_1d_mem (Memory-Optimized Version)

### Using Fragment Optimization

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_mul_relu_1d_mem(A, B, BLOCK_N: int):
    N = T.const("N")
    dtype = T.float16
    A: T.Tensor((N,), dtype)
    B: T.Tensor((N,), dtype)
    C = T.empty((N,), dtype)

    with T.Kernel(N // BLOCK_N, threads=256) as bx:
        # Step 1: Allocate register fragments
        A_frag = T.alloc_fragment((BLOCK_N,), dtype)
        B_frag = T.alloc_fragment((BLOCK_N,), dtype)
        C_frag = T.alloc_fragment((BLOCK_N,), dtype)
        
        # Step 2: Load from global memory to registers
        base_idx = bx * BLOCK_N
        T.copy(A[base_idx:base_idx + BLOCK_N], A_frag)
        T.copy(B[base_idx:base_idx + BLOCK_N], B_frag)
        
        # Step 3: Compute in registers
        for i in T.Parallel(BLOCK_N):
            result = A_frag[i] * B_frag[i]
            C_frag[i] = T.if_then_else(result > 0, result, T.float16(0))
        
        # Step 4: Write results back to global memory
        T.copy(C_frag, C[base_idx:base_idx + BLOCK_N])

    return C
```

### Fragment API Explanation

1. **T.alloc_fragment(shape, dtype)**
   - Allocate register fragment
   - shape: Fragment shape
   - dtype: Data type

2. **T.copy(src, dst)**
   - Copy data between different memory hierarchies
   - Global Memory → Fragment: `T.copy(A[start:end], frag)`
   - Fragment → Global Memory: `T.copy(frag, C[start:end])`

3. **Fragment Access**
   - Can access like normal array: `A_frag[i]`
   - Supports element-wise operations: `A_frag[i] * B_frag[i]`

### Optimization Rationale

**Unoptimized Version**:
- 3 global memory accesses per element

**Optimized Version**:
- Batch load/store, reduced memory transactions
- Intermediate computation in registers, faster

### Performance Improvement Sources

1. **Batch Memory Access**: Use instructions like ldg128 to load multiple elements at once
2. **Reduced Memory Transactions**: BLOCK_N elements only need 2 loads + 1 store
3. **Register Computation**: Intermediate results in registers, avoiding repeated global memory access
4. **Memory Coalescing**: Contiguous access more easily coalesced into efficient memory transactions by hardware
