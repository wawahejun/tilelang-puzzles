# Backward Op Implementation Guide

## Task 1: Implement tl_mul_relu_bcast

### Mathematical Formula

**Definition**: Given matrix $\mathbf{A} \in \mathbb{R}^{N \times M}$ and vector $\mathbf{b} \in \mathbb{R}^{M}$, compute:

$$C_{ij} = \max(0, A_{ij} \cdot b_j) \quad \forall i \in [0, N), j \in [0, M)$$

**Expanded Form**:

$$C_{ij} = \begin{cases} A_{ij} \cdot b_j & \text{if } A_{ij} \cdot b_j > 0 \\\\ 0 & \text{otherwise} \end{cases}$$

### Basic Version (One-to-one Correspondence with Mathematical Formula)

```python
@tilelang.jit
def tl_mul_relu_bcast(A, B, BLOCK_N: int, BLOCK_M: int):
    N, M = T.const("N, M")
    dtype = T.float16
    A: T.Tensor((N, M), dtype)
    B: T.Tensor((M,), dtype)
    C = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, M // BLOCK_M, threads=256) as (bx, by):
        n_idx = bx * BLOCK_N
        m_idx = by * BLOCK_M

        # ∀i ∈ [0, N), j ∈ [0, M)
        for i in T.Parallel(BLOCK_N):
            for j in T.Parallel(BLOCK_M):
                # Corresponds to mathematical formula: C_{ij} = max(0, A_{ij} · b_j)

                # Step 1: Calculate A_{ij} · b_j
                product = A[n_idx + i, m_idx + j] * B[m_idx + j]

                # Step 2: max(0, product)
                C[n_idx + i, m_idx + j] = T.if_then_else(
                    product > 0,
                    product,
                    T.float16(0)
                )

    return C
```

### Formula to Code Mapping

| Mathematical Symbol | Code Implementation |
|---------------------|---------------------|
| $\forall i \in [0, N)$ | `for i in T.Parallel(BLOCK_N)` |
| $\forall j \in [0, M)$ | `for j in T.Parallel(BLOCK_M)` |
| $A_{ij}$ | `A[n_idx + i, m_idx + j]` |
| $b_j$ | `B[m_idx + j]` |
| $A_{ij} \cdot b_j$ | `product = A[...] * B[...]` |
| $\max(0, \cdot)$ | `T.if_then_else(product > 0, product, 0)` |
| $C_{ij}$ | `C[n_idx + i, m_idx + j]` |

### Optimized Version (Using Fragment Cache)

```python
@tilelang.jit
def tl_mul_relu_bcast(A, B, BLOCK_N: int, BLOCK_M: int):
    N, M = T.const("N, M")
    dtype = T.float16
    A: T.Tensor((N, M), dtype)
    B: T.Tensor((M,), dtype)
    C = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, M // BLOCK_M, threads=256) as (bx, by):
        n_idx = bx * BLOCK_N
        m_idx = by * BLOCK_M
        A_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        B_local = T.alloc_fragment((BLOCK_M,), dtype)
        C_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)

        # Batch load data to registers
        T.copy(A[n_idx, m_idx], A_local)
        T.copy(B[m_idx], B_local)

        # Compute
        for i, j in T.Parallel(BLOCK_N, BLOCK_M):
            C_local[i, j] = A_local[i, j] * B_local[j]
            C_local[i, j] = T.if_then_else(C_local[i, j] > 0, C_local[i, j], 0)

        # Batch write back
        T.copy(C_local, C[n_idx, m_idx])

    return C
```

### Optimization Explanation

| Optimization Point | Basic Version | Optimized Version |
|--------------------|---------------|-------------------|
| Data loading | Read from global memory each computation | Batch load to registers |
| Memory access count | $O(N \cdot M \cdot 2)$ times | $O(N \cdot M + M)$ times |
| Loop structure | Nested loops | Combined 2D parallel loop |

## Task 2: Implement tl_mul_relu_bwd (Backward Propagation)

### Mathematical Formula

**Forward Pass**:

$$C_{ij} = \text{ReLU}(A_{ij} \cdot b_j) = \max(0, A_{ij} \cdot b_j)$$

**Backward Pass**: Given upstream gradient $\frac{\partial \mathcal{L}}{\partial C_{ij}} = dC_{ij}$, find $\frac{\partial \mathcal{L}}{\partial A_{ij}} = dA_{ij}$

**Chain Rule**:

$$\frac{\partial \mathcal{L}}{\partial A_{ij}} = \frac{\partial \mathcal{L}}{\partial C_{ij}} \cdot \frac{\partial C_{ij}}{\partial A_{ij}}$$

**Compute $\frac{\partial C_{ij}}{\partial A_{ij}}$**:

Let $z_{ij} = A_{ij} \cdot b_j$, then $C_{ij} = \text{ReLU}(z_{ij})$

$$\frac{\partial C_{ij}}{\partial z_{ij}} = \begin{cases} 1 & \text{if } z_{ij} > 0 \\\\ 0 & \text{otherwise} \end{cases}$$

$$\frac{\partial z_{ij}}{\partial A_{ij}} = b_j$$

Therefore:

$$\frac{\partial C_{ij}}{\partial A_{ij}} = \frac{\partial C_{ij}}{\partial z_{ij}} \cdot \frac{\partial z_{ij}}{\partial A_{ij}} = \mathbf{1}_{z_{ij} > 0} \cdot b_j$$

**Final Result**:

$$dA_{ij} = dC_{ij} \cdot b_j \cdot \mathbf{1}_{A_{ij} \cdot b_j > 0}$$

Where $\mathbf{1}_{condition}$ is the indicator function, taking 1 when condition is true, 0 otherwise.

### Basic Version (One-to-one Correspondence with Mathematical Formula)

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_mul_relu_bwd(A, B, dC, BLOCK_N: int, BLOCK_M: int):
    N, M = T.const("N, M")
    dtype = T.float16
    A: T.Tensor((N, M), dtype)
    B: T.Tensor((M,), dtype)
    dC: T.Tensor((N, M), dtype)
    dA = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, M // BLOCK_M, threads=256) as (pid_n, pid_m):
        n_idx = pid_n * BLOCK_N
        m_idx = pid_m * BLOCK_M

        # ∀i ∈ [0, N), j ∈ [0, M)
        for i in T.Parallel(BLOCK_N):
            for j in T.Parallel(BLOCK_M):
                # Corresponds to formula: dA_{ij} = dC_{ij} · b_j · 1_{A_{ij}·b_j > 0}

                # Step 1: Calculate z_{ij} = A_{ij} · b_j
                z = A[n_idx + i, m_idx + j] * B[m_idx + j]

                # Step 2: Calculate 1_{z > 0} (ReLU derivative)
                relu_grad = T.if_then_else(z > 0, T.float16(1), T.float16(0))

                # Step 3: dA_{ij} = dC_{ij} · b_j · 1_{z > 0}
                dA[n_idx + i, m_idx + j] = (
                    dC[n_idx + i, m_idx + j] * B[m_idx + j] * relu_grad
                )

    return dA
```

### Formula to Code Mapping

| Mathematical Symbol | Code Implementation |
|---------------------|---------------------|
| $\forall i \in [0, N)$ | `for i in T.Parallel(BLOCK_N)` |
| $\forall j \in [0, M)$ | `for j in T.Parallel(BLOCK_M)` |
| $z_{ij} = A_{ij} \cdot b_j$ | `z = A[...] * B[...]` |
| $\mathbf{1}_{z > 0}$ | `T.if_then_else(z > 0, 1, 0)` |
| $dC_{ij}$ | `dC[n_idx + i, m_idx + j]` |
| $b_j$ | `B[m_idx + j]` |
| $dA_{ij}$ | `dA[n_idx + i, m_idx + j]` |

### Optimized Version (Using Shared Memory + Fragment)

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_mul_relu_bwd(A, B, dC, BLOCK_N: int, BLOCK_M: int):
    N, M = T.const("N, M")
    dtype = T.float16
    A: T.Tensor((N, M), dtype)
    B: T.Tensor((M,), dtype)
    dC: T.Tensor((N, M), dtype)
    dA = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, M // BLOCK_M, threads=256) as (pid_n, pid_m):
        n_idx = pid_n * BLOCK_N
        m_idx = pid_m * BLOCK_M

        # Input data uses shared memory (needs multiple reads)
        A_local = T.alloc_shared((BLOCK_N, BLOCK_M), dtype)
        B_local = T.alloc_shared((BLOCK_M,), dtype)
        dC_local = T.alloc_shared((BLOCK_N, BLOCK_M), dtype)

        # Output data uses fragment (computed then written back directly)
        dA_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)

        # Batch load data
        T.copy(A[n_idx, m_idx], A_local)
        T.copy(B[m_idx], B_local)
        T.copy(dC[n_idx, m_idx], dC_local)

        # Compute
        for i, j in T.Parallel(BLOCK_N, BLOCK_M):
            dA_local[i, j] = (
                T.if_then_else(A_local[i, j] * B_local[j] > 0, 1, 0)
                * dC_local[i, j]
                * B_local[j]
            )

        # Batch write back
        T.copy(dA_local, dA[n_idx, m_idx])

    return dA
```

### Memory Allocation Strategy

| Variable | Storage Type | Purpose | Reason |
|----------|--------------|---------|--------|
| `A_local` | shared | Input | Needs multiple reads ($z$ calculation + condition check) |
| `B_local` | shared | Input | Needs multiple reads ($z$ calculation + $dA$ calculation) |
| `dC_local` | shared | Input | Only read once, shared memory for coalesced access |
| `dA_local` | fragment | Output | Computation result, written back directly |

### Why Use Shared Memory for Inputs?

1. **$b_j$ needs multiple reads**: $z_{ij} = A_{ij} \cdot b_j$ and $dA_{ij} = dC_{ij} \cdot b_j \cdot \mathbf{1}_{z > 0}$
2. **Shared memory has larger capacity**: Suitable for storing block-level data
3. **Memory coalesced access**: Batch loading is more efficient than element-wise access

## Summary

### Basic Version vs Optimized Version

| Aspect | Basic Version | Optimized Version |
|--------|---------------|-------------------|
| Purpose | Understand algorithm, corresponds to mathematical formula | Engineering optimization, improve performance |
| Data access | Direct global memory access | Preload to shared/fragment |
| Code structure | Element-wise operation, clear steps | Batch operation, combined computation |
| Applicable scenario | Learning, debugging | Production, performance-sensitive scenarios |
