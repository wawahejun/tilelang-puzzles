# Puzzle 04: Backward Op Code Analysis

## Overview

This puzzle introduces how to implement **backward propagation operators** in TileLang, helping understand how TileLang handles custom requirements, especially gradient computation in deep learning.

## What is Backward Propagation?

Backward propagation is the core algorithm for deep learning training, used to compute gradients of the loss function with respect to network parameters. According to the chain rule, we need to compute gradients layer by layer from the output layer to the input layer.

### Chain Rule Review

If `C = f(A)`, then the gradient of loss `L` with respect to `A` is:

```
dL/dA = dL/dC * dC/dA
```

Where `dL/dC` is the known upstream gradient (usually denoted as `dC`).

## 04-2: Backward Propagation Implementation

### Problem Definition

Now we need to compute the gradient of loss with respect to `A`, denoted `dA`, given the upstream gradient `dC`.

**Input**:
- `A: [N, M]` - Forward pass input tensor
- `B: [M,]` - Forward pass input vector
- `dC: [N, M]` - Gradient of loss with respect to C

**Output**:
- `dA: [N, M]` - Gradient of loss with respect to A

### Mathematical Derivation

Forward operation: $C = \text{ReLU}(A \cdot B)$

According to chain rule:

$$\frac{\partial L}{\partial A} = \frac{\partial L}{\partial C} \cdot \frac{\partial C}{\partial A}$$

Expanded as:

$$\frac{\partial L}{\partial A} = \frac{\partial L}{\partial C} \cdot \frac{\partial C}{\partial z} \cdot \frac{\partial z}{\partial A}$$

Where $z = A \cdot B$, substituting ReLU and its derivative:

$$\frac{\partial L}{\partial A} = dC \cdot \text{ReLU}'(A \cdot B) \cdot B$$

ReLU derivative is defined as:

$$\text{ReLU}'(x) = \begin{cases} 1 & \text{if } x > 0 \\\\ 0 & \text{if } x \leq 0 \end{cases}$$

Therefore:

$$\frac{\partial L}{\partial A_{ij}} = dC_{ij} \cdot B_j \cdot \mathbb{1}(A_{ij} \cdot B_j > 0)$$

Where $\mathbb{1}(\cdot)$ is the indicator function.

**Computation Definition**:
```python
for i in range(N):
    for j in range(M):
        dA[i, j] = dC[i, j] * B[j] * (A[i, j] * B[j] > 0)
```

### PyTorch Reference Implementation

```python
def ref_mul_relu_bwd(A: torch.Tensor, B: torch.Tensor, dC: torch.Tensor):
    A = A.clone()
    B = B.clone()
    A.requires_grad_(True)
    B.requires_grad_(True)
    C = torch.relu(A * B)
    C.backward(dC)
    return A.grad
```

## Core Concepts

### 1. Gradient Computation Flow

```
Forward Pass:  A, B  →  C = ReLU(A * B)
                 ↓
Backward Pass:  dC    →  dA = dC * B * ReLU'(A * B)
```

### 2. ReLU Derivative

```python
# ReLU derivative implementation
relu_grad = T.if_then_else(A[i,j] * B[j] > 0, 1.0, 0.0)
# Or more concisely
relu_grad = (A[i,j] * B[j] > 0)
```

### 3. Broadcasting in Backward Propagation

In forward propagation, `B` is broadcast to `A`'s shape. In backward propagation:
- `dA` has the same shape as `A` `[N, M]`
- `B` still needs to be broadcast for computation
