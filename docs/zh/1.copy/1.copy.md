# Puzzle 01: Copy 代码解析

## 概述

这是 TileLang 系列的第一个 puzzle，介绍了最基础的操作：**数据拷贝**。虽然看似简单，但理解 `T.copy` 的工作原理对于后续学习至关重要。

## 为什么 Copy 如此重要？

在 GPU 编程中，数据拷贝是最常见的操作之一：
- **全局内存到共享内存**: 加载数据进行计算
- **共享内存到全局内存**: 写回计算结果
- **寄存器之间的拷贝**: 中间计算结果传递

高效的拷贝操作是高性能 GPU 内核的基础。

## 问题定义

### Part 1: 串行 Copy

**输入**:
- `A: [N]` - 输入张量 (float16)

**输出**:
- `B: [N]` - 复制后的张量 (float16)

**计算定义**:
```python
for i in range(N):
    B[i] = A[i]
```

### Part 2: 多线程 Copy

与 Part 1 相同的问题定义，但要求利用多个线程并行执行。

### Part 3: 多线程块 Copy

**额外参数**:
- `BLOCK_N: int` - 每个线程块处理的元素数量

要求利用多个线程块并行处理不同数据段。

## 核心概念

### `T.copy` API

`T.copy` 是 TileLang 中最核心的操作之一，它封装了高效的数据拷贝逻辑。

#### 两种索引方式

| 方式 | 语法 | 适用场景 |
|------|------|----------|
| 起始索引 | `T.copy(A[start], local_buf)` | 目标是局部 buffer（自动推断） |
| 显式切片 | `T.copy(A[start:end], B[start:end])` | 目标是全局张量（必须显式） |

**自动推断条件**：
- ✅ 目标是 `T.alloc_fragment` 或 `T.alloc_shared` 分配的局部 buffer
- ❌ 目标是全局张量（没有形状信息，无法推断）

**示例**：
```python
# ✅ 自动推断：目标是局部 buffer
A_local = T.alloc_fragment((BLOCK_M, BLOCK_K), dtype)
T.copy(A[pid_m * BLOCK_M, k * BLOCK_K], A_local)

# ❌ 不能自动推断：目标是全局张量
# T.copy(A[start], B[start])  # 错误！

# ✅ 必须显式切片：全局张量之间复制
T.copy(A[start:end], B[start:end])
```

#### 自动优化

`T.copy` 会自动：
1. **并行化**: 利用线程块内的所有线程并行拷贝
2. **向量化**: 使用 128-bit 向量化加载/存储指令
3. **内存合并**: 优化全局内存访问模式

### `T.Kernel` 启动配置

```python
with T.Kernel(blocks, threads=N) as block_id:
    ...
```

| 参数 | 说明 |
|------|------|
| `blocks` | 线程块数量 |
| `threads` | 每个线程块的线程数 |
| `block_id` | 当前线程块的索引（从 0 开始） |

### GPU 并行层次

```
Grid (网格)
├── Block 0 ── Thread 0, Thread 1, ..., Thread N-1
├── Block 1 ── Thread 0, Thread 1, ..., Thread N-1
├── ...
└── Block M-1 ── Thread 0, Thread 1, ..., Thread N-1
```

**优化路径**：
```
串行 (1 block, 1 thread)
  ↓  增加 block 内线程数
多线程 (1 block, 128 threads)
  ↓  增加 block 数量
多 Block 并行 (M blocks, 128 threads each)
```

## 与 PyTorch 对比

| 特性 | TileLang `T.copy` | PyTorch `clone()` |
|------|-------------------|-------------------|
| 执行位置 | GPU 内核内 | 宿主端调用 |
| 并行化 | 自动 | 自动 |
| 向量化 | 自动 128-bit | 自动 |
| 灵活性 | 可嵌入复杂内核 | 独立操作 |
| 用途 | 内核内部数据传输 | 张量复制 |

1. **`T.copy` 智能推断**: 只需指定起始索引，自动根据目标大小推断范围
