# Vector Add 实现指南

## 任务1: 实现 tl_mul_relu_1d

### 参考 tl_add_1d 的结构

```python
@tilelang.jit
def tl_mul_relu_1d(A, B, BLOCK_N: int):
    N = T.const("N")
    A: T.Tensor((N,), T.float16)
    B: T.Tensor((N,), T.float16)
    C = T.empty((N,), T.float16)

    with T.Kernel(N // BLOCK_N, threads=256) as bx:
        base_idx = bx * BLOCK_N
        for i in T.Parallel(BLOCK_N):
            # 步骤1: 计算乘法
            result = A[base_idx + i] * B[base_idx + i]
            # 步骤2: 应用ReLU (max(0, result))
            C[base_idx + i] = T.if_then_else(result > 0, result, T.float16(0))

    return C
```

### 关键点

1. **Kernel结构**: 与加法完全相同
2. **乘法操作**: `A[i] * B[i]` 是元素级乘法
3. **ReLU实现**: `T.if_then_else(condition, true_value, false_value)`
   - condition: `result > 0`
   - true_value: `result`
   - false_value: `T.float16(0)` 或直接 `0`

## 任务2: 实现 tl_mul_relu_1d_mem (内存优化版本)

### 使用Fragment优化

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_mul_relu_1d_mem(A, B, BLOCK_N: int):
    N = T.const("N")
    dtype = T.float16
    A: T.Tensor((N,), dtype)
    B: T.Tensor((N,), dtype)
    C = T.empty((N,), dtype)

    with T.Kernel(N // BLOCK_N, threads=256) as bx:
        # 步骤1: 分配寄存器片段
        A_frag = T.alloc_fragment((BLOCK_N,), dtype)
        B_frag = T.alloc_fragment((BLOCK_N,), dtype)
        C_frag = T.alloc_fragment((BLOCK_N,), dtype)
        
        # 步骤2: 从全局内存加载到寄存器
        base_idx = bx * BLOCK_N
        T.copy(A[base_idx:base_idx + BLOCK_N], A_frag)
        T.copy(B[base_idx:base_idx + BLOCK_N], B_frag)
        
        # 步骤3: 在寄存器中进行计算
        for i in T.Parallel(BLOCK_N):
            result = A_frag[i] * B_frag[i]
            C_frag[i] = T.if_then_else(result > 0, result, T.float16(0))
        
        # 步骤4: 将结果写回全局内存
        T.copy(C_frag, C[base_idx:base_idx + BLOCK_N])

    return C
```

### Fragment API 说明

1. **T.alloc_fragment(shape, dtype)**
   - 分配寄存器片段
   - shape: 片段的形状
   - dtype: 数据类型

2. **T.copy(src, dst)**
   - 在不同内存层次间复制数据
   - 全局内存 → Fragment: `T.copy(A[start:end], frag)`
   - Fragment → 全局内存: `T.copy(frag, C[start:end])`

3. **Fragment访问**
   - 可以像普通数组一样访问: `A_frag[i]`
   - 支持元素级操作: `A_frag[i] * B_frag[i]`

### 优化原理

**未优化版本**:
- 每个元素3次全局内存访问

**优化版本**:
- 批量加载/存储，减少内存事务数
- 中间计算在寄存器中进行，速度更快

### 性能提升来源

1. **批量内存访问**: 使用ldg128等指令一次加载多个元素
2. **减少内存事务**: BLOCK_N个元素只需2次加载 + 1次存储
3. **寄存器计算**: 中间结果在寄存器中，避免重复访问全局内存
4. **内存合并**: 连续访问更容易被硬件合并成高效的内存事务
