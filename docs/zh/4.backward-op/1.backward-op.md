# Puzzle 04: Backward Op 代码解析

## 概述

这个 puzzle 介绍了如何在 TileLang 中实现**反向传播算子**，帮助理解 TileLang 如何处理自定义需求，特别是深度学习中的梯度计算。

## 什么是反向传播?

反向传播（Backpropagation）是深度学习训练的核心算法，用于计算损失函数对网络参数的梯度。根据链式法则，我们需要从输出层向输入层逐层计算梯度。

### 链式法则回顾

如果 `C = f(A)`，那么损失 `L` 对 `A` 的梯度为：

```
dL/dA = dL/dC * dC/dA
```

其中 `dL/dC` 是已知的上游梯度（通常记为 `dC`）。

## 04-1: 融合乘法 ReLU 与广播

### 问题定义

在实现反向传播之前，我们先定义前向操作：将 2D 张量 `A` 与 1D 向量 `B` 相乘，然后应用 ReLU 激活函数。

在数学描述和 PyTorch 参考实现中，`B` 会沿着 `A` 的行维度进行广播；而在 TileLang 里，内核中使用的是显式索引 `B[j]`，所以这里把广播语义直接体现在循环结构里。

**输入**:
- `A: [N, M]` - 2D 输入张量
- `B: [M,]` - 1D 输入向量

**输出**:
- `C: [N, M]` - 2D 输出张量

**计算定义**:
```python
for i in range(N):
    for j in range(M):
        C[i, j] = max(0, A[i, j] * B[j])
```

### PyTorch 参考实现

```python
def ref_mul_relu_bcast(A: torch.Tensor, B: torch.Tensor):
    # torch.mul 会自动广播 B 到 A 的形状
    return (A * B).relu_()
```

## 04-2: 反向传播实现

### 问题定义

现在我们需要计算损失对 `A` 的梯度 `dA`，已知上游梯度 `dC`。

**输入**:
- `A: [N, M]` - 前向传播的输入张量
- `B: [M,]` - 前向传播的输入向量
- `dC: [N, M]` - 损失对 `C` 的梯度

**输出**:
- `dA: [N, M]` - 损失对 `A` 的梯度

### 数学推导

前向操作: $C = \text{ReLU}(A \cdot B)$

根据链式法则：

$$\frac{\partial L}{\partial A} = \frac{\partial L}{\partial C} \cdot \frac{\partial C}{\partial A}$$

展开为：

$$\frac{\partial L}{\partial A} = \frac{\partial L}{\partial C} \cdot \frac{\partial C}{\partial z} \cdot \frac{\partial z}{\partial A}$$

其中 $z = A \cdot B$，代入 ReLU 及其导数：

$$\frac{\partial L}{\partial A} = dC \cdot \text{ReLU}'(A \cdot B) \cdot B$$

ReLU 的导数定义为：

$$\text{ReLU}'(x) = \begin{cases} 1 & \text{if } x > 0 \\\\ 0 & \text{if } x \leq 0 \end{cases}$$

因此：

$$\frac{\partial L}{\partial A_{ij}} = dC_{ij} \cdot B_j \cdot \mathbb{1}(A_{ij} \cdot B_j > 0)$$

其中 $\mathbb{1}(\cdot)$ 是指示函数。

**计算定义**:
```python
for i in range(N):
    for j in range(M):
        dA[i, j] = dC[i, j] * B[j] * (A[i, j] * B[j] > 0)
```

### PyTorch 参考实现

```python
def ref_mul_relu_bwd(A: torch.Tensor, B: torch.Tensor, dC: torch.Tensor):
    A = A.clone()
    B = B.clone()
    A.requires_grad_(True)
    B.requires_grad_(True)
    C = torch.relu(A * B)
    C.backward(dC)
    return A.grad
```

## 核心概念

### 1. 梯度计算流程

```
前向传播:  A, B  →  C = ReLU(A * B)
                 ↓
反向传播:  dC    →  dA = dC * B * ReLU'(A * B)
```

### 2. ReLU 导数

```python
# ReLU 导数的实现
relu_grad = T.if_then_else(A[i,j] * B[j] > 0, 1.0, 0.0)
# 或者更简洁地
relu_grad = (A[i,j] * B[j] > 0)
```

### 3. 广播在反向传播中的处理

在前向传播中，`B` 被广播到 `A` 的形状。在反向传播中：
- `dA` 的形状与 `A` 相同 `[N, M]`
- `B` 仍然需要广播参与计算
