# Puzzle 03: Outer Vector Add 代码解析

## 概述

这个puzzle介绍了如何在TileLang中实现**外向量加法**操作，从一维世界进入二维世界。

## 什么是外向量加法?

外向量加法（Outer Vector Add）是将两个一维向量组合成一个二维矩阵的操作：

```
A: [N,]  向量
B: [M,]  向量
C: [N, M]  矩阵

C[i, j] = A[i] + B[j]
```

### 直观理解

假设 `A = [1, 2, 3]`，`B = [10, 20]`：

```
        B[0]=10    B[1]=20
A[0]=1    11        21
A[1]=2    12        22
A[2]=3    13        23
```

每个矩阵元素 `C[i,j]` 是 `A[i]` 和 `B[j]` 的和。

### PyTorch 参考实现

```python
def ref_outer_add(A: torch.Tensor, B: torch.Tensor):
    return torch.add(input=A[:, None], other=B[None, :])
```

利用广播机制：`A[:, None]` 变成 `[N, 1]`，`B[None, :]` 变成 `[1, M]`，相加后得到 `[N, M]`。

## 核心概念

### 1. 二维张量处理

与之前的puzzle不同，这里输出 `C` 是二维张量：

```python
C = T.empty((N, M), dtype)  # 二维输出
```

### 2. 双重循环迭代

需要两个迭代器分别遍历 A 和 B：

```python
for i in range(N):      # 遍历 A
    for j in range(M):  # 遍历 B
        C[i, j] = A[i] + B[j]
```

### 3. Block 分块策略

由于输出是二维矩阵，需要考虑如何分块：

```python
with T.Kernel(N // BLOCK_N, M // BLOCK_M, threads=256) as (bx, by):
    # bx: 处理 A 的哪个块
    # by: 处理 B 的哪个块
```

## 与前一个 Puzzle 的区别

| 特性 | Puzzle 02 (Vector Add) | Puzzle 03 (Outer Add) |
|------|------------------------|----------------------|
| 输入维度 | 两个 1D 向量 | 两个 1D 向量 |
| 输出维度 | 1D 向量 | 2D 矩阵 |
| 计算模式 | 逐元素: `C[i] = A[i] + B[i]` | 外积式: `C[i,j] = A[i] + B[j]` |
| Block 配置 | 1D: `T.Kernel(N // BLOCK_N)` | 2D: `T.Kernel(N // BLOCK_N, M // BLOCK_M)` |
| 索引计算 | 单一索引 `base_idx + i` | 双重索引 `(i, j)` |

## 学习要点

1. **二维 Kernel 启动**: `T.Kernel(grid_x, grid_y)` 支持二维 grid
2. **多重 Block 索引**: `(bx, by)` 分别表示两个维度的 block 索引
3. **切片操作**: `A[start:end]` 和 `C[i_start:i_end, j_start:j_end]`
4. **数据重用**: A 的元素在每行重复使用，B 的元素在每列重复使用
