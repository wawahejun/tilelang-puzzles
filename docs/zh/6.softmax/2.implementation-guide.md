# Softmax 详细实现解析

本文档提供 Softmax 算子的完整实现解析，从基础版本到优化版本。

---

## 核心约束

Softmax 需要对每一行进行归约操作（max 和 sum），因此本文档中的教学实现采用如下划分：

```
┌────────────────────────────────────────────────────┐
│  Kernel 配置: T.Kernel(N // BLOCK_N)               │
│  ──────────────────────────────────────────────    │
│  △ 本文档不在 M 维度做跨块并行归并                 │
│  ✓ 可以在 N 维度分块 (行间独立)                    │
│  ✓ M 维度使用 T.Serial 串行分块处理                │
└────────────────────────────────────────────────────┘
```

这样做的目的不是说明 M 维并行不可能，而是为了把注意力集中在 softmax 的数值稳定和 online 更新逻辑上。

---

## 版本一：基础三阶段实现

按照 Softmax 的标准数学定义，分三个阶段逐步实现。

### 数学定义

对于输入 $A \in \mathbb{R}^{N \times M}$，输出 $B$ 为：

$$B_{i,j} = \frac{e^{A_{i,j} - m_i}}{\sum_{k=1}^{M} e^{A_{i,k} - m_i}}$$

其中 $m_i = \max_j A_{i,j}$ 为第 $i$ 行的最大值。

### 三阶段分解

为了数值稳定，我们将计算分为三个阶段：

```
┌─────────────────────────────────────────────────────────────┐
│  阶段 1: 计算 m_i = max_j(A_{i,j})                          │
│  阶段 2: 计算 s_i = sum_j(exp(A_{i,j} - m_i))               │
│  阶段 3: 计算 B_{i,j} = exp(A_{i,j} - m_i) / s_i            │
└─────────────────────────────────────────────────────────────┘
```

---

### 阶段 1：找每行的最大值

#### 数学公式

$$m_i = \max_{j=1}^{M} A_{i,j}$$

展开为循环形式：

$$m_i^{(0)} = -\infty$$

$$m_i^{(j+1)} = \max\left(m_i^{(j)}, A_{i,j}\right) \quad \forall j \in [0, M)$$

#### 代码实现

```python
# m_i^{(0)} = -∞
T.fill(max_val, -T.infinity(dtype))

# m_i^{(j+1)} = max(m_i^{(j)}, A_{i,j})
for m_blk_id in T.Serial(M // BLOCK_M):
    T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
    T.reduce_max(A_local, cur_max, dim=1, clear=True)
    
    for i in T.Parallel(BLOCK_N):
        max_val[i] = T.max(max_val[i], cur_max[i])
```

#### 公式与代码对照

| 数学公式 | 代码实现 |
|----------|----------|
| $m_i^{(0)} = -\infty$ | `T.fill(max_val, -T.infinity(dtype))` |
| $A_{i,j}$ | `A_local[i, j]` |
| $\max_{j}$ | `T.reduce_max(A_local, cur_max, dim=1)` |
| $m_i^{(j+1)} = \max(m_i^{(j)}, A_{i,j})$ | `max_val[i] = T.max(max_val[i], cur_max[i])` |

---

### 阶段 2：计算指数和

#### 数学公式

$$s_i = \sum_{j=1}^{M} e^{A_{i,j} - m_i}$$

展开为循环形式：

$$s_i^{(0)} = 0$$

$$s_i^{(j+1)} = s_i^{(j)} + e^{A_{i,j} - m_i} \quad \forall j \in [0, M)$$

#### 代码实现

```python
# s_i^{(0)} = 0
T.clear(sum_val)

# s_i^{(j+1)} = s_i^{(j)} + e^{A_{i,j} - m_i}
for m_blk_id in T.Serial(M // BLOCK_M):
    T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
    
    # e_{i,j} = e^{A_{i,j} - m_i} = 2^{(A_{i,j} - m_i)·log2(e)}
    for i, j in T.Parallel(BLOCK_N, BLOCK_M):
        B_local[i, j] = T.exp2((A_local[i, j] - max_val[i]) * log2_e)

    # s_i += sum_j(e_{i,j})
    T.reduce_sum(B_local, sum_val, dim=1, clear=False)
```

#### 公式与代码对照

| 数学公式 | 代码实现 |
|----------|----------|
| $s_i^{(0)} = 0$ | `T.clear(sum_val)` |
| $e^{A_{i,j} - m_i}$ | `T.exp2((A_local[i, j] - max_val[i]) * log2_e)` |
| $s_i^{(j+1)} = s_i^{(j)} + e^{A_{i,j} - m_i}$ | `T.reduce_sum(..., clear=False)` |

#### T.exp2 的数学基础
$$e^x = 2^{x\cdot\log_2(e)}$$

其中 $\log_2(e) \approx 1.44269504$。

```python
# e^{A_{i,j} - m_i} = 2^{log2(e) · (A_{i,j} - m_i)}
T.exp2((A_local[i, j] - max_val[i]) * log2_e)
```

---

### 阶段 3：归一化

#### 数学公式

$$B_{i,j} = \frac{e^{A_{i,j} - m_i}}{s_i}$$

#### 代码实现

```python
for m_blk_id in T.Serial(M // BLOCK_M):
    T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
    
    # B_{i,j} = e^{A_{i,j} - m_i} / s_i
    for i, j in T.Parallel(BLOCK_N, BLOCK_M):
        B_local[i, j] = T.exp2((A_local[i, j] - max_val[i]) * log2_e)
        B_local[i, j] = B_local[i, j] / sum_val[i]
    
    T.copy(B_local, B[pid_n * BLOCK_N, m_blk_id * BLOCK_M])
```

#### 公式与代码对照

| 数学公式 | 代码实现 |
|----------|----------|
| $e^{A_{i,j} - m_i}$ | `T.exp2((A_local[i, j] - max_val[i]) * log2_e)` |
| $/ s_i$ | `B_local[i, j] / sum_val[i]` |
| $B_{i,j}$ | `B[pid_n * BLOCK_N, m_blk_id * BLOCK_M]` |

---

### 完整代码

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_softmax_basic(A, BLOCK_N: int, BLOCK_M: int):
    log2_e = 1.44269504
    N, M = T.const("N, M")
    dtype = T.float32
    A: T.Tensor((N, M), dtype)
    B = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, threads=256) as pid_n:
        A_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        B_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        max_val = T.alloc_fragment((BLOCK_N,), dtype)
        sum_val = T.alloc_fragment((BLOCK_N,), dtype)
        cur_max = T.alloc_fragment((BLOCK_N,), dtype)
        
        # ========== 阶段 1: m_i = max_j(A_{i,j}) ==========
        # m_i^{(0)} = -∞
        T.fill(max_val, -T.infinity(dtype))
        
        # m_i^{(j+1)} = max(m_i^{(j)}, A_{i,j})
        for m_blk_id in T.Serial(M // BLOCK_M):
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
            T.reduce_max(A_local, cur_max, dim=1, clear=True)
            for i in T.Parallel(BLOCK_N):
                max_val[i] = T.max(max_val[i], cur_max[i])
        
        # ========== 阶段 2: s_i = sum_j(exp(A_{i,j} - m_i)) ==========
        # s_i^{(0)} = 0
        T.clear(sum_val)
        
        # s_i^{(j+1)} = s_i^{(j)} + e^{A_{i,j} - m_i}
        for m_blk_id in T.Serial(M // BLOCK_M):
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
            for i, j in T.Parallel(BLOCK_N, BLOCK_M):
                B_local[i, j] = T.exp2((A_local[i, j] - max_val[i]) * log2_e)
            T.reduce_sum(B_local, sum_val, dim=1, clear=False)
        
        # ========== 阶段 3: B_{i,j} = exp(A_{i,j} - m_i) / s_i ==========
        for m_blk_id in T.Serial(M // BLOCK_M):
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
            for i, j in T.Parallel(BLOCK_N, BLOCK_M):
                B_local[i, j] = T.exp2((A_local[i, j] - max_val[i]) * log2_e)
                B_local[i, j] = B_local[i, j] / sum_val[i]
            T.copy(B_local, B[pid_n * BLOCK_N, m_blk_id * BLOCK_M])

    return B
```

### 基础版本总结

| 阶段 | 数学公式 | 代码核心 |
|------|----------|----------|
| 1 | $m_i = \max_j A_{i,j}$ | `T.reduce_max` + `T.max` |
| 2 | $s_i = \sum_j e^{A_{i,j} - m_i}$ | `T.exp2` + `T.reduce_sum` |
| 3 | $B_{i,j} = e^{A_{i,j} - m_i} / s_i$ | `T.exp2` + `/` |

**内存访问**: 读取 A 三次

> 注：这里的基础版本与参考实现一样，输出缓冲区保持为 `float32`。如果教学上想引入“计算精度”和“写回精度”分离，可以在最后一步再显式 cast 到 `float16`。

## 版本二：Online Softmax（优化版本）

### 优化动机

基础版本需要读取 A 三次，内存带宽是性能瓶颈。观察发现阶段 1 和阶段 2 都遍历 A，可以合并。

**核心困难**：合并后，$m_i$ 会动态变化，需要不断调整之前累积的 $s_i$。

---

### LSE (Log-Sum-Exp) 技术

#### 数学定义

$$\text{LSE}(x_1, \ldots, x_n) = \log\left(\sum_{j=1}^{n} e^{x_j}\right)$$

#### 数值稳定形式

提取最大值 $m = \max_j x_j$：

$$\text{LSE} = m + \log\left(\sum_{j=1}^{n} e^{x_j - m}\right)$$

#### 关键性质

$$\text{softmax}(x_i) = e^{x_i - \text{LSE}}$$

**证明**：

$$e^{x_i - \text{LSE}} = e^{x_i - \log(\sum_j e^{x_j})} = \frac{e^{x_i}}{\sum_j e^{x_j}} = \text{softmax}(x_i)$$

---

### 块内调整公式（逐元素）

**场景**：处理单个元素时，最大值发生变化。

**数学推导**：

设当前累积指数和为 $s_{\text{old}}$（基于最大值 $m_{\text{old}}$），新元素 $A_j$ 使最大值变为 $m_{\text{new}}$。

需要调整：

$$s_{\text{new}} = \sum_{k < j} e^{A_k - m_{\text{new}}} = s_{\text{old}} \cdot e^{m_{\text{old}} - m_{\text{new}}}$$

加上新元素：

$$s_{\text{new}} = s_{\text{old}} \cdot e^{m_{\text{old}} - m_{\text{new}}} + e^{A_j - m_{\text{new}}}$$

---

### 块间 LSE 更新公式

**场景**：合并多个块的 LSE 结果。

设：
- $L_{\text{old}}$ = 之前所有块的累积 LSE
- $m_{\text{cur}}$ = 当前块的最大值
- $s_{\text{cur}}$ = 当前块的指数和（已减去 $m_{\text{cur}}$）

**数学推导**：

$$L_{\text{new}} = \log\left(e^{L_{\text{old}}} + \sum_{k \in \text{block}} e^{A_k}\right)$$

由于 $\sum_{k \in \text{block}} e^{A_k} = e^{m_{\text{cur}}} \cdot s_{\text{cur}}$：

$$L_{\text{new}} = \log\left(e^{L_{\text{old}}} + e^{m_{\text{cur}}} \cdot s_{\text{cur}}\right)$$

**与参考实现一致的写法**：提取 $m_{\text{cur}}$

$$L_{\text{new}} = m_{\text{cur}} + \log\left(e^{L_{\text{old}} - m_{\text{cur}}} + s_{\text{cur}}\right)$$

这正是参考实现中使用的两阶段 LSE 合并方式。它的优点是公式短、和代码一一对应：

```python
lse[i] = cur_max_A[i] * log2_e + T.log2(
    T.exp2(lse[i] - cur_max_A[i] * log2_e) + cur_sum_exp_A[i]
)
```

不过这里要避免一个常见误解：这个写法和参考实现**并不是在数学上最强的稳定形式**。如果当前块最大值明显小于历史块最大值，那么 `L_old - m_cur` 可能为正，指数项也可能变大。若要追求更稳健，一般会改用更强的锚点选择，如参考FlashAttention的风格写法。

**转换为 log2 空间**：

设 $\alpha = \log_2(e) \approx 1.44269504$，令 $L' = L \cdot \alpha$（即 $\log_2$ 空间中的 LSE，对应代码中的 `lse` 变量）：

$$L'_{\text{new}} = m_{\text{cur}} \cdot \alpha + \log_2\left(2^{L'_{\text{old}} - m_{\text{cur}} \cdot \alpha} + s_{\text{cur}}\right)$$

---

### 替代写法：FlashAttention 风格

LSE 方法使用对数运算统一表示，另一种直观的方法是直接维护全局最大值和全局指数和，这正是 FlashAttention 的核心思想。

#### 数学推导

设：
- $m_{\text{global}}$ = 所有已处理元素的全局最大值
- $d_{\text{global}}$ = 所有已处理元素的指数和（已减去 $m_{\text{global}}$）

当处理新块时，最大值从 $m_{\text{old}}$ 变为 $m_{\text{new}}$，需要缩放旧值：

$$d_{\text{global}}^{\text{new}} = d_{\text{global}}^{\text{old}} \cdot e^{m_{\text{old}} - m_{\text{new}}} + s_{\text{cur}}$$

**关键性质**：由于 $m_{\text{old}} \leq m_{\text{new}}$，故 $m_{\text{old}} - m_{\text{new}} \leq 0$，因此 $e^{m_{\text{old}} - m_{\text{new}}} \leq 1$，**无溢出风险**。

最终输出：

$$B_{i,j} = \frac{e^{A_{i,j} - m_{\text{global}}}}{d_{\text{global}}}$$

#### 两种写法对比

| 方面 | LSE 写法 | FlashAttention 写法 |
|------|----------|---------------------|
| 核心变量 | `lse` (单个值，和参考答案一致) | `m_global`, `d_global` (两个值) |
| 函数调用 | `T.log2()` + `T.exp2()` |  `T.exp2()` |
| 临时变量 | 无 | 需要 `m_prev` 保存旧最大值 |
| 代码直观性 | 数学抽象 | 直观易懂 |
| 最终输出 | `exp2(A - lse)` | `exp2(A - m) / d` |


#### 核心代码差异

**LSE 写法**（更新阶段）：

```python
# 单步更新，使用对数运算合并
for i in T.Parallel(BLOCK_N):
    lse[i] = cur_max_A[i] * log2_e + T.log2(
        T.exp2(lse[i] - cur_max_A[i] * log2_e) + cur_sum_exp_A[i]
    )
```

**FlashAttention 写法**（更新阶段）：

```python
# 分三步更新，无对数运算
for i in T.Parallel(BLOCK_N):
    m_prev = m_global[i]                                    # 保存旧值
    m_global[i] = T.max(m_global[i], cur_max_A[i])          # 更新最大值
    d_global[i] = d_global[i] * T.exp2((m_prev - m_global[i]) * log2_e)  # 缩放旧和

# ... 计算当前块的 cur_sum_exp_A ...

for i in T.Parallel(BLOCK_N):
    d_global[i] += cur_sum_exp_A[i]                         # 累加新和
```

**LSE 写法**（输出阶段）：

```python
for i, j in T.Parallel(BLOCK_N, BLOCK_M):
    B_local[i, j] = T.exp2(A_local[i, j] * log2_e - lse[i])
```

**FlashAttention 写法**（输出阶段）：

```python
for i, j in T.Parallel(BLOCK_N, BLOCK_M):
    B_local[i, j] = T.exp2((A_local[i, j] - m_global[i]) * log2_e) / d_global[i]
```

#### FlashAttention 风格完整代码

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_softmax_flashattn_style(A, BLOCK_N: int, BLOCK_M: int):
    log2_e = 1.44269504
    N, M = T.const("N, M")
    dtype = T.float32
    A: T.Tensor((N, M), dtype)
    B = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, threads=256) as pid_n:
        A_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        B_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        cur_exp_A = T.alloc_fragment([BLOCK_N, BLOCK_M], dtype)
        cur_max_A = T.alloc_fragment([BLOCK_N], dtype)
        cur_sum_exp_A = T.alloc_fragment([BLOCK_N], dtype)

        # 分开维护最大值和指数和
        m_global = T.alloc_fragment([BLOCK_N], dtype)  # 全局最大值
        d_global = T.alloc_fragment([BLOCK_N], dtype)  # 全局指数和

        T.fill(m_global, -T.infinity(dtype))
        T.clear(d_global)

        # ========== Pass 1: 在线更新 m_global 和 d_global ==========
        for m_blk_id in T.Serial(M // BLOCK_M):
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
            T.reduce_max(A_local, cur_max_A, dim=1, clear=True)

            # 步骤 1: 更新 m_global，并缩放 d_global
            for i in T.Parallel(BLOCK_N):
                m_prev = m_global[i]
                m_global[i] = T.max(m_global[i], cur_max_A[i])
                # m_prev - m_global <= 0，安全无溢出
                d_global[i] = d_global[i] * T.exp2((m_prev - m_global[i]) * log2_e)

            # 步骤 2: 计算当前块相对于全局最大值的指数
            for i, j in T.Parallel(BLOCK_N, BLOCK_M):
                cur_exp_A[i, j] = T.exp2((A_local[i, j] - m_global[i]) * log2_e)

            T.reduce_sum(cur_exp_A, cur_sum_exp_A, dim=1, clear=True)

            # 步骤 3: 累加到全局和
            for i in T.Parallel(BLOCK_N):
                d_global[i] += cur_sum_exp_A[i]

        # ========== Pass 2: 计算最终输出 ==========
        for m_blk_id in T.Serial(M // BLOCK_M):
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)

            for i, j in T.Parallel(BLOCK_N, BLOCK_M):
                B_local[i, j] = T.exp2((A_local[i, j] - m_global[i]) * log2_e) / d_global[i]

            T.copy(B_local, B[pid_n * BLOCK_N, m_blk_id * BLOCK_M])

    return B
```

### 完整代码（LSE 写法）

```python
@tilelang.jit(
    pass_configs={
        tilelang.PassConfigKey.TL_DISABLE_WARP_SPECIALIZED: True,
        tilelang.PassConfigKey.TL_DISABLE_TMA_LOWER: True,
    },
)
def tl_softmax(A, BLOCK_N: int, BLOCK_M: int):
    log2_e = 1.44269504  # α = log_2(e)
    N, M = T.const("N, M")
    dtype = T.float32
    A: T.Tensor((N, M), dtype)
    B = T.empty((N, M), dtype)

    with T.Kernel(N // BLOCK_N, threads=256) as pid_n:
        A_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        B_local = T.alloc_fragment((BLOCK_N, BLOCK_M), dtype)
        cur_exp_A = T.alloc_fragment([BLOCK_N, BLOCK_M], dtype)
        cur_max_A = T.alloc_fragment([BLOCK_N], dtype)      # m_cur
        cur_sum_exp_A = T.alloc_fragment([BLOCK_N], dtype)  # s_cur
        lse = T.alloc_fragment([BLOCK_N], dtype)            # L_old

        # L^{(0)} = -∞ (因为 log(0) = -∞)
        T.fill(lse, -T.infinity(dtype))

        # ========== Pass 1: 计算 LSE ==========
        for m_blk_id in T.Serial(M // BLOCK_M):
            # 加载当前块
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)
            
            # m_cur = max_j(A_{i,j})
            T.reduce_max(A_local, cur_max_A, dim=1, clear=True)

            # e_{i,j} = e^{A_{i,j} - m_cur} = 2^{log2(e)·A_{i,j} - log2(e)·m_cur}
            for i, j in T.Parallel(BLOCK_N, BLOCK_M):
                cur_exp_A[i, j] = T.exp2(A_local[i, j] * log2_e - cur_max_A[i] * log2_e)

            # s_cur = sum_j(e_{i,j})
            T.reduce_sum(cur_exp_A, cur_sum_exp_A, dim=1, clear=True)

            # lse_new = m_cur·log2(e) + log_2(2^{lse_old - m_cur·log2(e)} + s_cur)
            for i in T.Parallel(BLOCK_N):
                lse[i] = cur_max_A[i] * log2_e + T.log2(
                    T.exp2(lse[i] - cur_max_A[i] * log2_e) + cur_sum_exp_A[i]
                )

        # ========== Pass 2: 计算 softmax ==========
        for m_blk_id in T.Serial(M // BLOCK_M):
            T.copy(A[pid_n * BLOCK_N, m_blk_id * BLOCK_M], A_local)

            # B_{i,j} = 2^{log2(e)·A_{i,j} - lse}   (等价于 e^{A_{i,j} - LSE}，其中 lse = log2(e)·LSE)
            for i, j in T.Parallel(BLOCK_N, BLOCK_M):
                B_local[i, j] = T.exp2(A_local[i, j] * log2_e - lse[i])

            T.copy(B_local, B[pid_n * BLOCK_N, m_blk_id * BLOCK_M])

    return B
```

---

### 公式与代码详细对照（LSE 写法）

#### Pass 1: LSE 计算

| 数学公式 | 代码实现 |
|----------|----------|
| $L'^{(0)} = -\infty$ | `T.fill(lse, -T.infinity(dtype))` |
| $m_{\text{cur}} = \max_j A_{i,j}$ | `T.reduce_max(A_local, cur_max_A, dim=1)` |
| $e_{i,j} = e^{A_{i,j} - m_{\text{cur}}}$ | `T.exp2(A_local[i, j] * log2_e - cur_max_A[i] * log2_e)` |
| $s_{\text{cur}} = \sum_j e_{i,j}$ | `T.reduce_sum(cur_exp_A, cur_sum_exp_A, dim=1)` |
| $L'\_{\text{new}} = m\_{\text{cur}} \alpha + \log\_2(2^{L'\_{\text{old}} - m\_{\text{cur}} \alpha} + s\_{\text{cur}})$ | `lse[i] = cur_max_A[i] * log2_e + T.log2(T.exp2(lse[i] - cur_max_A[i] * log2_e) + cur_sum_exp_A[i])` |

#### Pass 2: Softmax 输出

| 数学公式 | 代码实现 |
|----------|----------|
| $B_{i,j} = 2^{\log_2(e) \cdot A_{i,j} - lse}$ | `T.exp2(A_local[i, j] * log2_e - lse[i])` |
| $B_{i,j}$ | `B[pid_n * BLOCK_N, m_blk_id * BLOCK_M]` |

---

### 公式与代码详细对照 (FlashAttention 写法)

#### Pass 1: 在线更新 m_global 和 d_global

| 数学公式 | 代码实现 |
|----------|----------|
| $m^{(0)} = -\infty$ | `T.fill(m_global, -T.infinity(dtype))` |
| $d^{(0)} = 0$ | `T.clear(d_global)` |
| $m_{\text{cur}} = \max_j A_{i,j}$ | `T.reduce_max(A_local, cur_max_A, dim=1)` |
| $m_{\text{prev}} = m_{\text{old}}$ | `m_prev = m_global[i]` |
| $m_{\text{new}} = \max(m_{\text{old}}, m_{\text{cur}})$ | `m_global[i] = T.max(m_global[i], cur_max_A[i])` |
| $d_{\text{scaled}} = d_{\text{old}} \cdot e^{m_{\text{prev}} - m_{\text{new}}}$ | `d_global[i] = d_global[i] * T.exp2((m_prev - m_global[i]) * log2_e)` |
| $e_{i,j} = e^{A_{i,j} - m_{\text{new}}}$ | `T.exp2((A_local[i, j] - m_global[i]) * log2_e)` |
| $s_{\text{cur}} = \sum_j e_{i,j}$ | `T.reduce_sum(cur_exp_A, cur_sum_exp_A, dim=1)` |
| $d_{\text{new}} = d_{\text{scaled}} + s_{\text{cur}}$ | `d_global[i] += cur_sum_exp_A[i]` |

#### Pass 2: Softmax 输出

| 数学公式 | 代码实现 |
|----------|----------|
| $B_{i,j} = e^{A_{i,j} - m_{\text{global}}} / d_{\text{global}}$ | `T.exp2((A_local[i, j] - m_global[i]) * log2_e) / d_global[i]` |
| $B_{i,j}$ | `B[pid_n * BLOCK_N, m_blk_id * BLOCK_M]` |

## 版本对比

| 方面 | 基础版本 | Online Softmax (LSE) | Online Softmax (FlashAttention) |
|------|----------|----------------------|--------------------------------|
| 数学形式 | $m_i$, $s_i$ 分别计算 | LSE 统一表示 | $m$, $d$ 分开维护 |
| 遍历次数 | 3 次 | 2 次 | 2 次 |
| 内存访问 | 读 A 三次 | 读 A 两次 | 读 A 两次 |
| 函数调用 | `T.exp2` | `T.log2` + `T.exp2` |  `T.exp2` |
| 额外变量 | 无 | 无 | `m_prev` |
| 代码复杂度 | 简单 | 中等 | 中等 |
| 直观性 | 高 | 低（数学抽象） | 高（直接理解） |

## 总结

### 学习路径

```
基础版本: 数学公式 → 代码实现 → 一一对照
    │
    │  问题: 为什么要三次遍历？
    │
    ▼
优化动机: 合并阶段 1 和 2 → max 动态变化
    │
    │  解决方案
    │
    ├─→ LSE 写法: 数学优雅，对数统一表示
    │
    └─→ FlashAttention 写法: 自然直观
```

### 核心要点

| 概念 | 数学 | 代码 |
|------|------|------|
| 数值稳定性 | $e^{x - m}$ 其中 $m = \max(x)$ | `T.exp2((x - max_val) * log2_e)` |
| 最大值归约 | $m_i = \max_j A_{i,j}$ | `T.reduce_max(..., dim=1)` |
| 指数求和 | $s_i = \sum_j e^{A_{i,j} - m_i}$ | `T.reduce_sum(..., dim=1)` |
| LSE 更新 | $lse_{\text{new}} = m_{\text{cur}} \cdot \alpha + \log_2(2^{lse_{\text{old}} - m_{\text{cur}} \cdot \alpha} + s_{\text{cur}})$ | `lse[i] = cur_max_A[i] * log2_e + T.log2(...)` |
| FlashAttention 更新 | $d_{\text{new}} = d_{\text{old}} \cdot e^{m_{\text{old}} - m_{\text{new}}} + s_{\text{cur}}$ | `d_global[i] = d_global[i] * T.exp2((m_prev - m_global[i]) * log2_e) + cur_sum_exp_A[i]` |
| 最终输出 (LSE) | $B_{i,j} = 2^{\log_2(e) \cdot A_{i,j} - lse}$ | `T.exp2(A_local[i, j] * log2_e - lse[i])` |
| 最终输出 (FlashAttention) | $B_{i,j} = e^{A_{i,j} - m} / d$ | `T.exp2((A_local[i, j] - m_global[i]) * log2_e) / d_global[i]` |

补充两点和参考实现相关的工程假设：

- 本文所有完整代码默认 `N % BLOCK_N == 0` 且 `M % BLOCK_M == 0`，省略 tail/mask 逻辑。
- 输出缓冲区保持 `float32`，这样能把关注点集中在 online softmax 本身；若要讲 mixed precision，可在最终写回时再单独讨论 cast。
